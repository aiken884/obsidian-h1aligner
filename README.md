# H1Aligner

> **Align your note filename with the first H1 on file switch.**
> H1 即檔名 — Obsidian 筆記命名自動化外掛

🚧 **Phase 1 MVP under active development.** Full README lands in Epic E5.

## What it does (TL;DR)

When you switch to a markdown file in Obsidian, H1Aligner reads the first H1 (`# Title`) in the file and renames the file on disk to match. Quiet by default — no notices, no surprises.
